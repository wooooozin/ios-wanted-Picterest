# 🖥 프로젝트 소개
### **애플리케이션 설명**
- 서버 API를 이용하여 이미지를 받아와, 가변 세로 길이의 레이아웃으로 나타냅니다.
- 원하는 사진을 선택해 저장하는 기능을 갖습니다.
- 첫번째 Image List Page에서 별 버튼을 누르면 메모 작성과 함께 사진 저장 및 다시 별을 누르면 삭제됩니다.
- 두번째 Saved Imaged Page에서 저장된 사진을 확인하고 Long PressGeture로 삭제할 수 있습니다.
<br>
